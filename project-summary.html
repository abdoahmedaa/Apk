🚀 How to Run the App

Prerequisites: Android Studio, Java 17 ✅, Android SDK

Steps:

  1. Open Android Studio
  2. Select "Open an existing project"
  3. Navigate to the FocusFlow directory
  4. Wait for Gradle sync to complete
  5. Connect an Android device or start an emulator
  6. Click the Run button (▶️) or press Shift + F10
# Alternative: Command line build cd FocusFlow ./gradlew assembleDebug ./gradlew installDebug

🎨 App Screens

The app includes 5 main screens accessible via bottom navigation:

  • 🏠 Dashboard: Daily overview and quick stats
  • ✅ Tasks: Task management with filters and priorities
  • 📝 Todos: Multiple todo lists with progress tracking
  • 🎯 Habits: Habit tracking with streaks and statistics
  • ⚙️ Settings: App configuration and data management

💾 Database Schema

The app uses Room database with 5 main entities:

  • Tasks: Task management with priorities and due dates
  • TodoLists: Container for todo items
  • TodoItems: Individual todo items with completion status
  • Habits: Habit definitions with frequency and targets
  • HabitLogs: Daily habit completion tracking
